# <a name="translate-learning-pathways-custom-playlists"></a>Übersetzen von benutzerdefinierten Wiedergabelisten für Lernpfade
Bei Lernpfaden hängt die Art und Weise, wie Sie mit benutzerdefinierten Wiedergabelisten arbeiten, davon ab, wie Sie Lernpfade einrichten.  

- Wenn Sie eine neue mehrsprachige Website für Lernpfade bereitgestellt haben, können Sie eine neue Wiedergabeliste kopieren oder erstellen und die Wiedergabeliste in die von der Website unterstützten Sprachen übersetzen.
- Wenn Sie eine vorhandene Lernpfadwebsite mit bereits vorhandenen benutzerdefinierten Wiedergabelisten aktualisiert haben, können Sie die vorhandenen Wiedergabelisten in die von Ihrer Website unterstützten Sprachen übersetzen. Sie können auch neue Wiedergabelisten kopieren und erstellen. 

> [!IMPORTANT]
> - Von Microsoft bereitgestellte Wiedergabelisten sind bereits übersetzt und können nur geändert werden, wenn Sie eine Kopie der Wiedergabeliste erstellen. 

## <a name="create-or-copy-a-playlist-and-have-it-translated"></a>Erstellen oder Kopieren einer Wiedergabeliste und Übersetzen der Wiedergabeliste
Wenn Sie eine neue mehrsprachige Website für Lernpfade bereitgestellt oder die Website mit dem mehrsprachigen Web part aktualisiert haben, können Sie eine benutzerdefinierte Wiedergabeliste kopieren oder erstellen und die Wiedergabelisten- und Wiedergabelistenressourcen in die von Ihrer Website unterstützten Sprachen übersetzen. 

### <a name="copy-a-playlist-and-have-it-translated"></a>Kopieren einer Wiedergabeliste und Übersetzen der Wiedergabeliste
Um zu veranschaulichen, wie Sie eine neue benutzerdefinierte Wiedergabeliste erstellen und übersetzen können, verwenden wir das Feature Wiedergabelisten kopieren von Lernpfaden.

1.  Klicken Sie im Menü Start der **Lernpfade** auf **Verwaltung von Lernpfaden.** 
2.  Wählen **Sie unter Microsoft 365-Lernpfade** die Option **Erste** Tage aus, und wählen Sie dann Sechs einfache Schritte – Willkommen bei **Microsoft 365 aus.** 
3. Klicken **Sie auf Wiedergabeliste** kopieren, scrollen Sie auf der Seite nach unten, und klicken Sie **auf Detail bearbeiten.**   
4. Ändern Sie den Titel der Wiedergabeliste. In diesem Beispiel wird der Titel der Wiedergabeliste in **Fünf einfache Schritte geändert.** Die verfügbaren Sprachen hängen von den unterstützten Sprachen ab, die für die Website ausgewählt wurden. Um weitere Sprachen hinzuzufügen, müssen Sie der Website weitere unterstützte Sprachen hinzufügen.   
5. Klicken **Sie auf Details speichern.** 
6. Klicken Sie auf eine Sprache rechts neben der Standardsprache (in diesem Fall Englisch), um die Details der kopierten Wiedergabeliste anzuzeigen. Die Details für die neue Sprache spiegeln die ursprüngliche kopierte Wiedergabeliste wider und erfordern eine Übersetzung.
7. Notieren Sie sich, um den Übersetzer über die zu übersetzende Wiedergabelistendetails zu informieren.  

![Beispiel für italienische Seite.](media/custom_update_ml_pldetail.png)

> [!IMPORTANT]
> Übersetzerbenachrichtigungen sind nicht in benutzerdefinierte Wiedergabelisten integrierte. Übersetzer müssen manuell benachrichtigt werden. Übersetzer übersetzen die Kopien der Standardsprachendetails manuell in die angegebene Sprache. Sie müssen den Übersetzer über die übersetzungen benachrichtigen, die für Wiedergabelistendetails erforderlich sind. Es wird empfohlen, alle Wiedergabelistenänderungen, einschließlich hinzufügen, Bearbeiten oder Löschen von Wiedergabelistenressourcen, fertig zu stellen und dann den Übersetzer über die erforderlichen Übersetzungen für Wiedergabelistendetails und -objekte zu informieren.

## <a name="create-multilingual-assets-in-a-custom-playlist"></a>Erstellen mehrsprachiger Objekte in einer benutzerdefinierten Wiedergabeliste
Alle Von Microsoft bereitgestellten Objekte in einer Wiedergabeliste sind nicht bearbeitbar und erfordern keine Übersetzung. Von Microsoft bereitgestellte Ressourcen stehen in den von der Website unterstützten Sprachen zur Verfügung. Für benutzerdefinierte Objekte, die Sie erstellen, oder vorhandene benutzerdefinierte Objekte können Sie die von der Website unterstützten Sprachen hinzufügen.  Sehen wir uns an, wie Sprachen für neue und vorhandene benutzerdefinierte Objekte in einer Wiedergabeliste hinzugefügt werden. 

### <a name="create-a-new-multilingual-asset-for-a-playlist"></a>Erstellen einer neuen mehrsprachigen Ressource für eine Wiedergabeliste
In diesen Anweisungen wird davon ausgegangen, dass Sie die Wiedergabeliste "Fünf einfache Schritte" im Beispiel in diesem Thema verwenden. 

**So erstellen Sie das Objekt**
1. Wählen Sie für die Wiedergabeliste "Fünf einfache **Schritte"** unter Assets die Option **+ Neues Objekt aus,** und füllen Sie dann die Details für das Objekt aus. 
   - Geben **Sie für Titel**"Willkommen bei Microsoft 365 bei Contoso" ein. 
   - Wählen **Sie für Technologie** Microsoft **365 aus.**
![Formular, in dem Sie Objektdetails eingeben.](media/custom_ml_pl_createasset.png)
2. Wählen **Sie Asset erstellen** aus, und wählen Sie dann Asset speichern **aus.**

**So öffnen Sie das Objekt**
1. Wählen Sie das erstellte Objekt aus, und klicken Sie dann auf **Öffnen**.  
2. Nehmen Sie die Änderungen an der Seite wie gewünscht vor. Vorerst können Sie den Titel der Seite ändern und der Seite text hinzufügen, wie im folgenden Beispiel gezeigt, und dann auf **Veröffentlichen klicken.** 
3. Klicken Sie **auf Übersetzung**.
4. Wählen **Sie Erstellen** für die Sprache aus, die Sie möchten. In diesem Beispiel wählen wir Italienisch aus.
5. Wählen Sie **Ansicht** aus. Je nach den unterstützten Sprachen sollte nun eine Seite ähnlich dem folgenden Beispiel angezeigt werden.
![Beispielseite für Wiedergabelistenressourcen.](media/custom_ml_pl_createasset.png)
6. Klicken Sie auf **Veröffentlichen**.
7. Kopieren Sie für die erstellte Übersetzungsseite die URL aus dem Browser, und kehren Sie zur Seite Verwaltung der Lernpfade zurück.

**So fügen Sie eine Sprache für das erstellte Übersetzungsinventar hinzu** 
1. Kehren Sie zur Seite Verwaltung von Lernpfaden zurück, und wählen Sie das Bearbeitungssymbol (Bleistift) für das von Ihnen geöffnete Objekt "Fünf einfache Schritte" aus. 
2. Klicken **Sie auf Sprache** hinzufügen, und wählen Sie dann eine Sprache aus. In diesem Beispiel haben wir Italienisch ausgewählt. 
3. Geben Sie die URL der übersetzten Seite ein, die Sie erstellt haben. Es sollte eine Seite wie im folgenden Beispiel angezeigt werden. 
![Formular, in dem Sie die URL für eine bestimmte Sprachseite eingeben](media/custom_ml_pl_translateasseturl.png)
4. Klicken **Sie auf Asset speichern**.

## <a name="add-languages-for-existing-custom-playlists-and-assets"></a>Hinzufügen von Sprachen für vorhandene benutzerdefinierte Wiedergabelisten und Ressourcen
Wenn Sie über aktualisierte Lernpfade für mehrsprachigen Support verfügen und über benutzerdefinierte Wiedergabelisten und Ressourcen verfügen, können Sie unterstützte Sprachen für die Wiedergabelisten und Ressourcen hinzufügen. In diesen Anweisungen wird davon ausgegangen, dass Sie über eine benutzerdefinierte Wiedergabeliste verfügen, der Sie Sprachen hinzufügen können. Wenn Sie über keine benutzerdefinierte Wiedergabeliste verfügen, lesen Sie weiter oben in diesem Thema "Erstellen oder Kopieren einer Wiedergabeliste". 

> [!IMPORTANT]
> - Von Microsoft bereitgestellte Wiedergabelisten sind bereits übersetzt und können nur geändert werden, wenn Sie eine Kopie der Wiedergabeliste erstellen. Sie können nur Sprachen für benutzerdefinierte Wiedergabelisten hinzufügen. 
> - Zum Hinzufügen von Sprachen für benutzerdefinierte Wiedergabelisten und Ressourcen müssen mehrsprachiger Support und Sprachen aktiviert sein. Weitere Informationen finden Sie unter [Festlegen von Spracheinstellungen](./custom_update_ml.md#set-language-settings).

![Beispiel für eine benutzerdefinierte Wiedergabeliste.](media/custom_ml_pl_customselect.png)

4. Wählen **Sie Sprache hinzufügen** aus, und wählen Sie dann eine Sprache aus.  
5. Wählen **Sie Details speichern aus.**
 Die verfügbaren Sprachen hängen von den unterstützten Sprachen ab, die für die Website ausgewählt wurden. Um weitere Sprachen hinzuzufügen, müssen Sie der Website weitere unterstützte Sprachen hinzufügen.   
5. Klicken Sie auf eine Sprache rechts neben der Standardsprache (in diesem Fall Englisch), um die Details der kopierten Wiedergabeliste anzuzeigen. Die Details für die neue Sprache spiegeln die ursprüngliche kopierte Wiedergabeliste wider und erfordern eine Übersetzung.
![Beispieldetailsepage für Italienisch.](media/custom_ml_pl_editdetailit.png) 
7. Notieren Sie sich, um den Übersetzer über die zu übersetzende Wiedergabelistendetails zu informieren.  

**So öffnen Sie das Objekt**
1. Wählen Sie das erstellte Objekt aus, und klicken Sie dann auf **Öffnen**.  
2. Nehmen Sie die Änderungen an der Seite wie gewünscht vor. Vorerst können Sie den Titel der Seite ändern und der Seite text hinzufügen, wie im folgenden Beispiel gezeigt, und dann auf **Veröffentlichen klicken.** 
3. Klicken Sie **auf Übersetzung**.
4. Wählen **Sie Erstellen** für die Sprache aus, die Sie möchten. In diesem Beispiel wählen wir Italienisch aus.
5. Wählen Sie **Ansicht** aus. Je nach den unterstützten Sprachen sollte nun eine Seite ähnlich dem folgenden Beispiel angezeigt werden.

## <a name="notify-the-translator"></a>Benachrichtigen des Übersetzers
Wenn Sie mit dem Erstellen von Übersetzungsressourcen fertig sind, benachrichtigen Sie den Übersetzer über die erforderlichen Übersetzungen. Der Übersetzer hat:
- Übersetzen Sie Wiedergabelistendetails.
- Übersetzen Von Objektdetails.
- Übersetzt hinzugefügte Sprachseiten für ein Objekt.
- Benachrichtigen des Ansers über die Übersetzungen, dass die Übersetzungen zur Überprüfung bereit sind

## <a name="add-languages-for-an-existing-custom-playlist"></a>Hinzufügen von Sprachen für eine vorhandene benutzerdefinierte Wiedergabeliste
Wenn Sie über aktualisierte Lernpfade für mehrsprachigen Support verfügen und über benutzerdefinierte Wiedergabelisten und Ressourcen verfügen, können Sie unterstützte Sprachen für die Wiedergabelisten und Ressourcen hinzufügen. 

> [!IMPORTANT]
> - Von Microsoft bereitgestellte Wiedergabelisten sind bereits übersetzt und können nur geändert werden, wenn Sie eine Kopie der Wiedergabeliste erstellen. Sie können nur Sprachen für benutzerdefinierte Wiedergabelisten hinzufügen. 
> - Zum Hinzufügen von Sprachen für benutzerdefinierte Wiedergabelisten und Ressourcen müssen mehrsprachiger Support und Sprachen aktiviert sein. Weitere Informationen zum Aktivieren von mehrsprachigem Support und Optionen finden Sie unter [Set Language Settings](./custom_update_ml.md#set-language-settings). 


## <a name="instructions-for-the-translator"></a>Anweisungen für den Übersetzer
Wenn Sie Wiedergabelisten Sprachen hinzufügen, müssen Sie den Übersetzer über die Änderungen benachrichtigen. Sie können den Übersetzer auf Folgendes verweisen: 

### <a name="translate-playlist-details"></a>Übersetzen von Wiedergabelistendetails
Klicken Sie im Menü Start **von** Lernpfaden auf Verwaltung **von Lernpfaden.** 
1. Klicken Sie auf die benutzerdefinierte Wiedergabeliste, die eine Übersetzung erfordert, und klicken Sie dann auf die Sprachen. 
2. Klicken **Sie auf Detail bearbeiten,** erstellen Sie die Übersetzungen für die Wiedergabeliste, und klicken Sie dann auf 
3. Klicken **Sie auf Details speichern.** 
4. Benachrichtigen Sie den Übersetzungsanforderungsgeber, dass die Übersetzung abgeschlossen ist. 

### <a name="translate-asset-details"></a>Übersetzen von Objektdetails
Klicken Sie im Menü Start **von** Lernpfaden auf Verwaltung **von Lernpfaden.** 
1. Klicken Sie auf die benutzerdefinierte Wiedergabeliste, für die eine Übersetzung erforderlich ist. 
2. Scrollen Sie auf der Seite nach unten, und wählen Sie dann unter Objekte die Option Bearbeiten für das Objekt aus, das Sie bearbeiten möchten, und wählen Sie dann die Sprache aus. 
3. Nehmen Sie die Übersetzungen für das Objekt vor, und klicken Sie dann auf **Asset speichern.**  

### <a name="translate-the-added-language-page-for-the-asset"></a>Übersetzen der hinzugefügten Sprachseite für das Objekt
Klicken Sie im Menü Start **von** Lernpfaden auf Verwaltung **von Lernpfaden.** 
1. Klicken Sie auf die benutzerdefinierte Wiedergabeliste, für die eine Übersetzung erforderlich ist. 
2. Scrollen Sie auf der Seite nach unten, wählen Sie dann unter Objekte das Objekt aus, wählen Sie die Sprache aus, und klicken Sie dann auf Öffnen. 
3. Erstellen Sie die Übersetzungen für die Seite, und klicken Sie dann auf **Veröffentlichen**.
